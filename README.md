# s3-multipart-upload

### Description
[Work In Progress] A script takes parts of a file, uploads them to AWS S3 using the multipart uploading function and combines the parts into one object upon completion.

### Execution
Run `s3multiUploader.sh`

### Environment
Ubuntu 18.04 LTS
